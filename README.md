# 📝 NotesApp (MVVM + Room + Coroutines)

A simple Notes App built using **MVVM architecture**, **Room Database**, and **Coroutines** in Kotlin.

## 🔧 Features

- Add a new note
- Delete notes
- Local data persistence using Room
- MVVM architecture with ViewModel and LiveData
- Simple and clean UI
